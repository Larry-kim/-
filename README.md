**문제설명**   
   
정수 num이 짝수일 경우 "Even"을 반환하고 홀수인 경우 "Odd"를 반환하는 함수, solution을 완성해주세요.<br>
   
**제한조건**
* num은 int 범위의 정수입니다.
* 0은 짝수입니다.

**입출력 예**
|num|return|
|------|---|
|3|"Odd"|
|4|"Even"|

**제한조건**
* N의 범위 : 100,000,000 이하의 자연수

**입출력 예**
|N|answer|
|------|---|
|123|6|
|987|24|
